Compare css background size property contain and cover

分別比較在限定寬度比圖片小、限定寬度和圖片一樣大、限定寬度比圖片大，三種情況下css background-size:contain、background-size:cover呈現狀況。

background和img不一樣, background如果沒有用高+寬撐開，他是不占空間;相對，image就算不設寬高，也會占空間

用Chrome測試
參考:https://developer.mozilla.org/zh-CN/docs/Web/Guide/CSS/Scaling_background_images

範例1:width與圖片大小一樣

background-size:contain

background-size:cover



範例2:width比圖片小

background-size:contain

background-size:cover



範例3:width比圖片大

background-size:contain

background-size:cover
